Dias_Isabela_Tasks

We explore the main concepts of topological data analysis in visualization.

  • Dias_Isabela_Tasks.html presents additional explanations and reasoning to solve the tasks and the solution for Task 2.

  • readmetask1.html provides the readme for Task 1.

  • readmetask3.html provides the readme for the Task 3.

  • hydrogen.pyis the Script for Task 1 .

  • hydrogen.vtkis the dataset for Task 1.

  • Task3.py is the Script for Task 3.

  • 2d_scalar_field.csv is the dataset for Task 3.

[!NOTE] The hydrogen.py script is compatible with both Windows and macOS operating systems. However, we recommend running the Task3.py script on a macOS system.
