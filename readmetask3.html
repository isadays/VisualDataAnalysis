Iso-contours Generation and Critical Point Analysis

This script generates isocontours and analyses the critical points from a 2D scalar field dataset using ParaView’s Python scripting and TTK (Topological Data Analysis Toolkit) filters.

Prerequisites

  • ParaView (Version 5.10.1)
  • Python (ParaView’s Python interpreter is used)
  • Pandas
  • VTK
  • Topological Data Analysis Toolkit Plugin (TTK)

Usage

  1. Open a terminal.

  2. Run the script with the following command on the Dias_Isabela_Tasks directory:

pvpython code.py <csv_file> <isovalue_1> <isovalue_2> <isovalue_3> <output_filename>

Note that the pvpython is required to run the script. Then, make sure that you are using the correct path. For instance:

 /Applications/ParaView-5.10.1.app/Contents/bin/pvpython Task3.py '2d_scalar_field.csv' 0 1 1.9 'isocontours.png'

  1. Replace <csv_file>, <isovalue_1>,<isovalue_2>, <isovalue_3>, and with the appropriate values.

  2. The script will generate isocontours and analyze critical points. Results will be saved as CSV files and an image file.

Parameters

  • <csv_file> : Path to the CSV file containing the 2D scalar field data.

  • <isovalue_1>, <isovalue_2>, <isovalue_3>: Isovalues for generating contours.

  • <output_filename>: Path to the output png file.

Output

  • Image file: output_filename.png The generated isocontours saved as a png file. ( Dias_Isabela_Tasks folder)
  • Output files: scalarfieldcroutput.csv and scalarfieldcroutput.vtk containing the outputs of ttk::ScalarFieldCriticalPoints Filter and scalarfieldcroutputClass.csv file containing the classification of the critical points (TTK_Outputs folder)

Script Details

The script utilizes ParaView’s Python scripting interface to perform isocontour generation and critical point analysis. TTK (Topological Data Analysis Toolkit) filters are employed for analyzing critical points. Critical points are classified into categories such as Local Minimum, Local Maximum, Saddle Type 1, Saddle Type 2, Degenerate, Regular, or Unknown.
