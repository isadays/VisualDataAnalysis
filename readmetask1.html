Hydrogen Isosurfaces with VTK

This Python scripts utilizes the Visualization Toolkit (VTK) library to visualize isosurfaces extracted from a 3D volume dataset. The dataset consists of the probability density for the 3d electron position in a hydrogen atom.

Dataset information

The dataset is a structured points dataset stored in a VTK file. This type of dataset represents 3D image data where the points are arranged in a regular grid. This grid is defined in terms of dimensions (64,64,64), origin (0,0,0), spacing (1,1,1).

  • File Format: VTK
  • Dataset Type: Structured Points
  • Dimensions: W,H, D (width, height and depth)
  • Scalar values: Probability density

Isosurface extraction

  • The isosurface extraction is performed using the vtkContourFilter. In this script we set the initial isovalue as 0.03.

Visualization components

  • Outline: The outline of the dataset is created using the vtkOutlineFilter. This provides a visual representation of the volume boundaries.

  • Isosurfaces: The isosurface extracted from the dataset is visualized using the vtkPolyDataMapper and vtkActor. The color of the isosurface is determined by the vtkColorTransferFunction, which maps scalar values to colors.

  • Scalar Bar: A scalar bar is added to the visualization scene using vtkScalarBarActor. This bar provides a color reference for the iso-surfaces, indicating the mapping between scalar values and colors.

Requirements

  • Python 3
  • VTK library

Usage

  1. Ensure you have Python installed on your system.

2.Install the VTK library using the following command:

 pip install vtk
  1. Run the script using Python:
python hydrogen.py
  1. The same result is obtained by using Paraview (pvpython)
pvpython hydrogen.py
  1. The script will display a 3D visualization window. The animation can be reinitialized by double-clicking within the 3D visualization window.

[!NOTE] The script is compatible with both Windows and MacOS operating systems.
